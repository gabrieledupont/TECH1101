HTML Cheat Sheet

Tag Description Example
<html> The root element of an HTML page. <html> <head> </head> <body> </body> </html>
<head> Contains metadata about the HTML page. <head> <title>Page Title</title> </head>
<title> Defines the title of the HTML page, which is displayed in the browser's title bar. <title>Page Title</title>
<body> Contains the visible content of the HTML page. <body> <h1>Heading</h1> <p>Paragraph</p> </body>
<h1> to <h6> Defines headings of different sizes, from largest (h1) to smallest (h6). <h1>Heading</h1>
<p> Defines a paragraph of text. <p>Paragraph</p>
<a> Defines a hyperlink to another web page or resource. <a href="https://example.com">Link</a>
<img> Defines an image to be displayed on the page. <img src="image.jpg" alt="Image">
<ul> and <li> Defines an unordered list of items. <ul> <li>Item 1</li> <li>Item 2</li> &
