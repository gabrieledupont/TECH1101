JavaScript Cheat Sheet

JavaScript Cheat Sheet

Concept Description Example
Variables Used to store data or values in memory for later use. var x = 10;
Data Types JavaScript has several data types, including strings, numbers, booleans, and objects. var str = "Hello"; var num = 10; var bool = true; var obj = {};
Functions Reusable blocks of code that perform a specific task. function addNumbers(a, b) { return a + b; }
Conditional Statements Used to perform different actions based on different conditions. if (x > 10) { console.log("x is greater than 10"); }
Loops Used to repeat a block of code multiple times. for (var i = 0; i < 10; i++) { console.log(i); }
Arrays Used to store multiple values in a single variable. var arr = [1, 2, 3, 4, 5];
Objects Used to store multiple values and functions as properties of a single variable. var person = { name: "John", age: 30, sayHello: function() { console.log("Hello!"); } };
DOM Manipulation Used to modify HTML and CSS on a web page dynamically. document.getElementById("myElement").innerHTML = "New text";
Events Actions that occur in response to user interaction or other triggers. document.getElementById("myButton").addEventListener("click", function() { console.log("Button clicked"); });
